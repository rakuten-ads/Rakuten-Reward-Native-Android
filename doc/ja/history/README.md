# 更新履歴

### バージョン 8.0.0
<hr/>
リリース日: 2026/01/02

* **改善されたトークン管理のために`RewardTokenProvider`を導入**
    * トークン有効期限の自動処理 - `TOKENEXPIRED`ステータスの手動チェックが不要
    * SDK API呼び出し前に`ONLINE`ステータスを待つ必要がなくなりました
    * `setRidToken()`および`setRaeToken()` APIを非推奨化（引き続きサポートされますが、移行を推奨）
    * 詳細については[移行ガイド](../migration/v8-migration.md)を参照してください
* **MemberInfo API呼び出しタイミングの変更**
    * `onCreate()`でのみ呼び出されるようになりました（以前は`onStart()`で呼び出し）
    * バックエンドAPIトラフィックの削減
    * ユーザーデータ（ポイント/未獲得数）を表示する場合、`onStart()`で手動で`memberInfo()`を呼び出す必要があります
* Android 16 (API 36) のサポート
* Javaプロジェクトでの統合を容易にするため、Java互換のコールバックインターフェースを提供する`rewardnativesdk-java`モジュールを導入しました
